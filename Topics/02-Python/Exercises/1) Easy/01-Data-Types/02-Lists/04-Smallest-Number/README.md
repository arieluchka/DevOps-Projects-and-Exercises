# Find the Smallest Number in a List

Write a Python program to get the smallest number from a list.

## Examples:

### Example 1:

**Input**: [2, 8, 6]

**Output**: 2

### Example 2:

**Input**: [34, -22, 0, 45]

**Output**: -22

### Example 3:

**Input**: [1.5, 0.3, 2.2, 0.1]

**Output**: 0.1
