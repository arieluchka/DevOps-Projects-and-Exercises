# Remove Duplicates from a List

Write a Python program to remove duplicates from a list.

## Examples:

### Example 1:

**Input**: [1, 2, 2, 3, 4, 4, 4]

**Output**: [1, 2, 3, 4]

### Example 2:

**Input**: ['apple', 'banana', 'apple', 'orange']

**Output**: ['apple', 'banana', 'orange']

### Example 3:

**Input**: [10, 10, 20, 20, 30, 30, 30, 40]

**Output**: [10, 20, 30, 40]
