# Calculate the length of a string

Write a Python program to multiply all the items in a list.

## Examples:
### Example 1:

**Input**: [4, 1, 2]

**Output**: 8

### Example 2:

**Input**: [1, 2, 3, 4, -2]

**Output**: -48

### Example 3:

**Input**: [10, -10, 10, -10]

**Output**: 10000
