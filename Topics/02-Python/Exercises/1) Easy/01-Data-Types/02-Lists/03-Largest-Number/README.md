# Find the Largest Number in a List

Write a Python program to get the largest number from a list.

## Examples:

### Example 1:

**Input**: [4, 1, 8, 3]

**Output**: 8

### Example 2:

**Input**: [10, 320, 45, 60]

**Output**: 320

### Example 3:

**Input**: [-1, -3, -10, -2]

**Output**: -1
